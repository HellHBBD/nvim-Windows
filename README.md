## Install

### Windows

```
md %LOCALAPPDATA%\nvim
cd %LOCALAPPDATA%\nvim
git clone https://github.com/HellHBBD/nvim-config.git .
```

### Linux

```
mkdir -p ~/.config/nvim
cd ~/.config/nvim
git clone https://github.com/HellHBBD/nvim-config.git .
```

## Uninstall

### Windows

```
rd %LOCALAPPDATA%\nvim
```

### Linux

```
rm -rf ~/.config/nvim
```
