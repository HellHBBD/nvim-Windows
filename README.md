## Install config files

### Windows

```
mkdir C:\Users\user\AppData\Local\nvim\
cd C:\Users\user\AppData\Local\nvim\
git clone https://github.com/HellHBBD/nvim-config.git .
```

### Linux

```
mkdir ~/.config
mkdir ~/.config/nvim
cd ~/.config/nvim
git clone https://github.com/HellHBBD/nvim-config.git .
```
